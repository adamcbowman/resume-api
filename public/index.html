GET request to /api returns full JSON resume.
GET request to /api/basics returns basic information section.
GET request to /api/skills returns skills list.

About this Resume

The purpose of this resume is to demonstrate the use of the JSON Resume format to create a resume file, and build an API to serve it.
The resume is created using the JSON Resume Schema.

The data for this resume lives in a GitHub gist file

It is served by an express API deployed on Heroku.
